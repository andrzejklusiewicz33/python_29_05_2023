Aby pobrać sobie ten kod do lokalnego katalogu i móc go łatwo aktualizować, powinieneś/powinnaś wykonać poniższe czynności:
1. Stworzyć folder w którym ma znaleźć się kod
2. Uruchomić w tym folderze konsolę git bash - jest łatwo dostępna w menu kontekstowym pod prawym przyciskiem myszy
3. Wykonać poniższą komendę w konsoli git bash w stworzonym katalogu:

git clone https://github.com/andrzejklusiewicz33/python_29_05_2023.git

4. Po wykonaniu tej komendy, wszystko co znajduje się w tym repo zostanie skopiowane do nowego folderu na Twoim dysku.
Jeśli zaktualizuję pliki, a Ty będziesz chciał/chciała je zaktualizować również u siebie, wystarczy że w konsoli git bash w tym 
folderze przejdziesz do nowo utworzonego folderu i uruchomisz komendę git'a:

cd python_29_05_2023
git pull
